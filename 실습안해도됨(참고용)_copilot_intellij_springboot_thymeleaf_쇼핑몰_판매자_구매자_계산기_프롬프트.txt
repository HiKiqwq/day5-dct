19. application.properties 설정 프롬프트
text
Create application.properties with:

1. Server settings:
   server.port=8080
   server.servlet.context-path=/

2. Thymeleaf settings:
   spring.thymeleaf.cache=false (for development)
   spring.thymeleaf.prefix=classpath:/templates/
   spring.thymeleaf.suffix=.html
   spring.thymeleaf.encoding=UTF-8

3. Session settings:
   server.servlet.session.timeout=30m
   server.servlet.session.tracking-modes=cookie

4. Logging:
   logging.level.com.shopping=DEBUG
   logging.pattern.console=%d{yyyy-MM-dd HH:mm:ss} - %msg%n

5. File upload settings (for product images):
   spring.servlet.multipart.max-file-size=10MB
   spring.servlet.multipart.max-request-size=10MB
