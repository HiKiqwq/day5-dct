2. 구매자 회원가입 화면 프롬프트
text
Create a Java Swing JPanel for buyer sign-up with the following components:
- JTextField for ID (with duplicate check button)
- JPasswordField for password and confirm password
- JTextField for name, email, phone number, birthdate
- JButton for submit and cancel
- Add validation for password matching and email format
- Use GridBagLayout for proper alignment
- Add Korean/Chinese comments for each component
3. 상품 관리 화면 프롬프트
text
Create a Java Swing JPanel for seller product management with:
- JTable showing product list (code, name, category, price, stock, status)
- JPanel with search field and category filter
- JButtons for product registration, modification, deletion
- JButton for inventory management
- Add product registration dialog with JDialog
- Include product image upload functionality using JFileChooser
4. 주문 관리 화면 프롬프트
text
