Java Swing 购物商城管理系统（买家/卖家双模式）

核心要求
不使用数据库（DB）：所有数据使用 ArrayList 等集合类在内存中管理（模拟数据）。

文件结构：

主程序：ShoppingMallApp.java

买家功能：ShoppingBuyer.java

卖家功能：ShoppingSeller.java

计算器：Cal.java

计算器功能：在任何界面点击“计算器”按钮，都会弹出一个新的独立窗口（JFrame）。

🛒 买家功能详细说明
1. 注册功能
组件：ID、密码、确认密码、姓名、邮箱、电话、生日

验证：

ID 重复检查

密码与确认密码一致

邮箱格式验证

必填字段检查

2. 会员信息查看
显示：ID、姓名、邮箱、电话、注册日期、最后登录时间、等级/积分

3. 信息修改
可修改：密码、邮箱、电话、地址

需验证当前密码

4. 会员注销
确认复选框

密码验证

注销原因下拉框（服务不满意、使用频率低、隐私顾虑、其他）

详细原因输入框

警告提示

5. 收货地址管理
列表显示：地址名称、收件人、地址、电话、是否默认

操作按钮：新增、修改、删除、设为默认

新增/修改表单：地址名称、收件人、邮编、基本地址、详细地址、电话、设为默认复选框

6. 订单记录
列表显示：订单号、日期、商品名、数量、金额、状态、是否取消

筛选功能：日期范围、状态（全部/已支付/准备发货/发货中/已完成/已取消）

排序：最新/最旧/金额高/金额低

订单详情：基本信息、配送信息、商品列表、支付信息、状态变更（管理员）、取消订单（买家）

🏪 卖家功能详细说明
1. 仪表盘
今日订单数、总销售额、月度销售图表（Graphics2D）

新订单通知列表、库存不足商品列表、最新评论

统计：今日访问量、新会员数、平均订单金额

2. 商品管理
列表：商品代码、名称、分类、价格、库存、状态、注册日期

搜索：关键词 + 分类下拉

操作：注册、修改、删除、库存管理、Excel 导出

注册/修改表单：名称、分类、价格、库存、折扣率、描述、图片上传、状态、选项管理

3. 订单管理
列表：订单号、日期、客户名、商品、金额、支付方式、状态

筛选：日期、状态、支付方式

操作：详情查看、批量状态变更、取消处理、发货单打印、Excel 导出

详情：基本信息、商品列表、配送信息、支付信息、处理历史、状态变更按钮

4. 配送管理
列表：订单号、日期、客户名、商品、地址、邮编

操作：输入运单号、选择快递公司、变更配送状态、添加备注、批量处理

5. 库存管理
列表：商品代码、名称、当前库存、安全库存、状态、入库日期

筛选：分类、库存状态（充足/不足/缺货）

操作：选择商品、输入变更数量、变更原因、入库/出库/调整、设置库存警报、Excel 导出

6. 销售管理
统计图表：日/周/月销售图表

查询：日期范围

详细表格：日期、订单数、总销售额、平均订单金额、各支付方式销售额

指标：总销售额、总订单数、平均订单金额、退款金额

操作：Excel 报告、图表导出

7. 评论管理
列表：评论编号、商品名、作者、评分、内容、日期、状态

筛选：评分（1-5星）、日期范围、状态（全部/未回复/已回复）

详情：商品名、作者、评分、内容、图片

操作：回复、删除、处理举报

8. 公告/活动管理
公告列表：编号、标题、日期、浏览量、重要性

公告表单：标题、重要性（一般/重要/紧急）、内容、附件、公告期限、置顶复选框

活动管理：活动列表、活动注册/修改、优惠券发放

9. 会员管理
列表：会员编号、ID、姓名、邮箱、注册日期、等级、状态

搜索：关键词 + 条件（ID/姓名/邮箱/电话）

详情：基本信息、订单记录、积分、等级

操作：状态变更（正常/休眠/停用/注销）、等级变更、积分调整

🚀 编译与运行
命令行编译
bash
# 编译所有 Java 文件
javac ShoppingMallApp.java ShoppingBuyer.java ShoppingSeller.java Cal.java model/*.java util/*.java

# 运行主程序
java ShoppingMallApp
IDE 运行
在 IntelliJ IDEA 或 Eclipse 中，右键点击 ShoppingMallApp.java，选择 "Run as Java Application"。
